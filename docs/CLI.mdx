## `lambda`

```
$ ./gccli lambda --help
```

```
NAME:
   gccli lambda - manage lambda functions

USAGE:
   gccli lambda command [command options] [arguments...]

COMMANDS:
   init     generates a new lambda with the data provided
   info     show information of current lambda
   lang     list all available languages for lambdas
   start    starts function image to test locally
   logs     display logs of lambda
   stop     stop lambda container running
   edit     change name/description of current lambda
   invoke   invoke a remote function
   result   gets result of task
   deploy   builds and pushes multi-arch container images
   public   create public endpoint to invoke directly
   list     list all Lambdas
   delete   deletes lambda function
   restore  restore a lambda

OPTIONS:
   --help, -h  show help
```
