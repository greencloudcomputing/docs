### Introduction

Run `gccli` command to make sure you have the app installed.

<cliWindow>

```text
$ ./gccli
NAME:
   gccli - the future of clean, ethical cloud computing....

USAGE:
   gccli [global options] command [command options] [arguments...]

VERSION:
   23.03.08

COMMANDS:
   login   you must login to start using GreenCloud
   logout  to stop using GreenCloud simply logout
   node    manage node settings
   info    get info from user
   status  get all running lambdas
   lambda  manage lambda functions
   update  check if a newer version is available

GLOBAL OPTIONS:
   --help, -h     show help
   --version, -v  print the version

## `lambda`

```
$ ./gccli lambda --help
```

```
NAME:
   gccli lambda - manage lambda functions

USAGE:
   gccli lambda command [command options] [arguments...]

COMMANDS:
   init     generates a new lambda with the data provided
   info     show information of current lambda
   lang     list all available languages for lambdas
   start    starts function image to test locally
   logs     display logs of lambda
   stop     stop lambda container running
   edit     change name/description of current lambda
   invoke   invoke a remote function
   result   gets result of task
   deploy   builds and pushes multi-arch container images
   public   create public endpoint to invoke directly
   list     list all Lambdas
   delete   deletes lambda function
   restore  restore a lambda

OPTIONS:
   --help, -h  show help
```

### `lambda init`

```
$ ./gccli lambda init --help
```

```
NAME:
   gccli lambda init - generates a new lambda with the data provided

USAGE:
   gccli lambda init -n NAME -l LANG -d 'DESCRIPTION' -p

OPTIONS:
   --lang value, -l value         language or template to use
   --name value, -n value         name to be used in lambda
   --description value, -d value  description of lambda
   --offline, -o                  create lambda in offline mode (default: false)
   --no-folder, --nf              use current folder (default: false)
   --help, -h                     show help
```

### `lambda start`

```
$ ./gccli lambda start --help
```

```

OPTIONS:

   --timeout value, -t value  maximum execution time in seconds (default: 10)
